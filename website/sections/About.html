🎯 What We Do

Chaos Kitten is the first agentic security scanner that uses advanced AI reasoning to discover API vulnerabilities that traditional scanners miss.

Unlike static scanners that rely on known signatures, Chaos Kitten thinks like an attacker – chaining multiple techniques, understanding business logic, and exploiting complex edge cases.

🧠
LLM-Powered Reasoning: Claude 3.5 Sonnet analyzes your OpenAPI spec and crafts intelligent attacks
💥
Chaos Mode: Boundary testing, type confusion, Unicode attacks, and crash hunting
🔓
Full Automation: MFA bypass, auth flows, rate limiting, and business logic testing

🔬 How It Works

Chaos Kitten combines AI reasoning with battle-tested security techniques:

  1. Parse: OpenAPI spec → endpoint map + schemas
  2. Plan: LLM generates attack strategies per endpoint
  3. Execute: Fuzzing + chaos payloads + exploit chains
  4. Analyze: Response anomalies → severity scoring
  5. Report: SARIF/JUnit/HTML with remediation advice

📈 Why Chaos Kitten?

Traditional scanners fail because:

  • They miss business logic flaws
  • Can't chain multiple attack techniques
  • Struggle with MFA/auth flows
  • Generate too many false positives

Chaos Kitten succeeds because:

  • AI understands your API context
  • Chaos Mode finds crash-only bugs
  • Zero false positives with reasoning
  • CI/CD ready with SARIF output
